# 📒 Address Book Application (Python)

## 📌 Project Overview
The **Address Book Application** is a Python console-based project used to manage contacts.  
This project demonstrates the use of **Object-Oriented Programming (OOP), File Handling, Collections, and Data Serialization**.

The system allows users to **create, edit, delete, search, sort, and store contacts** in multiple formats like **TXT, CSV, and JSON**.

---

## ✨ Features

### 👤 UC1 – Create Contact
Create a contact with the following details:
- First Name
- Last Name
- Address
- City
- State
- Zip Code
- Phone Number
- Email

### ➕ UC2 – Add Contact
Add a new contact to the address book using console input.

### ✏️ UC3 – Edit Contact
Edit an existing contact using the person's name.

### ❌ UC4 – Delete Contact
Delete a contact using the person's name.

### 📚 UC5 – Add Multiple Contacts
Add multiple contacts to the address book using collections.

### 📂 UC6 – Multiple Address Books
Create multiple address books using a dictionary structure.

### 🚫 UC7 – Prevent Duplicate Entries
Prevent duplicate contacts based on the person's name.

### 🔎 UC8 – Search by City or State
Search contacts across address books by city or state.

### 📍 UC9 – View Persons by City or State
Display all contacts belonging to a specific city or state.

### 🔢 UC10 – Count Contacts by City or State
Display the number of contacts in a particular city or state.

### 🔤 UC11 – Sort by Person Name
Sort contacts alphabetically by person name.

### 🗂️ UC12 – Sort by City, State, or Zip
Sort contacts based on:
- City
- State
- Zip Code

### 📄 UC13 – File IO
Store and read contacts using **text files**.

### 📊 UC14 – CSV File Handling
Store and read contacts using **CSV format**.

### 🧾 UC15 – JSON File Handling
Store and read contacts using **JSON format**.
